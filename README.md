# 🦷 OdontoFlow

OdontoFlow é um sistema fullstack para gestão de clínicas odontológicas, criado com base em um problema real: substituir sistemas antigos por uma solução moderna, escalável e eficiente.

O projeto tem como objetivo centralizar operações da clínica, melhorar o fluxo de atendimento e trazer mais controle sobre o negócio.

---

## 📌 Sobre o Projeto

O OdontoFlow nasceu a partir da necessidade real de modernização de sistemas utilizados em clínicas odontológicas.

A proposta é desenvolver uma aplicação completa, cobrindo:

- Gestão de pacientes
- Controle de agenda
- Histórico clínico
- Fluxo financeiro
- Organização operacional da clínica

---

## 🧠 Objetivo

Criar uma solução moderna que substitua sistemas legados, focando em:

- Performance
- Escalabilidade
- Boa experiência de uso
- Arquitetura sólida
- Aplicação de boas práticas de engenharia de software

---

## ⚙️ Stack Tecnológica

### Backend
- Java 17+
- Spring Boot
- Spring Data
- Spring Security (planejado)
- API REST

### Frontend
- React
- TailwindCSS

### Banco de Dados
- SQL (modelagem relacional)
- DynamoDB (estudo e possível uso para escalabilidade)

### Cloud
- AWS (deploy e infraestrutura)

---

## 🏗️ Arquitetura

O projeto segue arquitetura em camadas:


Controller → Service → Repository → Entity → Database


Com foco em:

- Separação de responsabilidades
- Código limpo
- Facilidade de manutenção
- Escalabilidade futura
